Watcha

Second Projet réalisé au sein de la Wild Code School, de avril à mai 2019. Watcha, réalisé en équipe de 5 sur une période de deux mois utilise une API permettant de récupérer des élèments de films (BA, synopsis etc.). Il permet de retrouver un film, via un moteur de recherche, par son titre; cliquer sur un acteur renvoi à la liste de ses films; les films peuvent être ajoutés à une liste de favoris, liste que l'on peut retrouver dans le menu "liste".

Un lien dans le footer présente succintement l'équipe de developpement.

Outre l'API, Watcha a été développé en JavaScript et React.JS.
